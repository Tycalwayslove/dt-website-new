# 注册功能实现说明

## 功能概述

已成功实现用户注册功能，包括完整的表单验证、短信验证码获取和用户注册流程。

## 实现内容

### 1. API 接口配置 (`src/api/auth.ts`)

添加了以下接口和类型定义：

- **注册接口**: `apiRegister(payload: RegisterRequest)`
  - 接口地址: `/api/register`
  - 请求方法: POST
  - 请求参数:
    - `phone`: 手机号（必填）
    - `code`: 验证码（必填）
    - `password`: 密码（必填）
    - `confirm_password`: 确认密码（必填）
    - `agent_id`: 代理ID（选填）

- **获取验证码接口**: `apiGetSmsCode(payload: GetSmsCodeRequest)`
  - 接口地址: `/api/get_sms_code`
  - 请求方法: POST
  - 请求参数:
    - `phone`: 手机号
    - `type`: 验证码类型（默认为 'register'）

### 2. 注册页面 (`src/pages/Register.vue`)

基于 Element Plus 实现的注册页面，包含以下功能：

#### 表单字段
- 手机号输入框（带格式验证）
- 验证码输入框（带获取验证码按钮和60秒倒计时）
- 密码输入框（带显示/隐藏功能）
- 确认密码输入框（带一致性校验）
- 代理ID输入框（可选填）

#### 表单验证规则
- **手机号**:
  - 非空校验
  - 格式校验（1开头，11位数字）
  
- **验证码**:
  - 非空校验
  - 长度校验（4-6位）

- **密码**:
  - 非空校验
  - 长度校验（6-20位）

- **确认密码**:
  - 非空校验
  - 与密码一致性校验

#### 功能特性
- 获取验证码功能（60秒倒计时防重复点击）
- 实时表单验证
- 友好的错误提示
- 加载状态提示
- 注册成功后自动跳转到首页

### 3. 路由配置 (`src/router/index.ts`)

添加了注册页面路由：
- 路径: `/register`
- 路由名称: `Register`
- 页面标题: `注册 - 喵呜AI - 顾问式电商平台`

### 4. 登录弹窗更新 (`src/components/auth/LoginDialog.vue`)

在登录弹窗的账号登录区域添加了"立即注册"链接：
- 移动端和PC端都已更新
- 点击后跳转到注册页面

## 使用方法

### 访问注册页面
- 直接访问: `http://your-domain/register`
- 从登录弹窗点击"立即注册"链接

### 注册流程
1. 输入手机号
2. 点击"获取验证码"按钮
3. 输入收到的验证码
4. 设置密码
5. 确认密码
6. （可选）输入代理ID
7. 点击"注册"按钮
8. 注册成功后自动跳转到首页

## 样式说明

注册页面采用现代化设计：
- 响应式布局，支持移动端和桌面端
- 渐变背景色（绿色到蓝色）
- 圆角卡片设计
- 品牌色（#9BFF2B）主题按钮
- Element Plus 组件统一风格

## 注意事项

1. **验证码接口**: 实际的验证码发送接口需要后端配置，接口地址为 `/api/get_sms_code`

2. **注册接口**: 后端注册接口地址为 `/api/register`，需要确保接口返回标准格式：
```json
{
  "status": "ok" | "error",
  "message": "提示信息",
  "data": null
}
```

3. **代理ID字段**: 为可选字段，如果不需要可以留空

4. **密码要求**: 当前设置为6-20位，可根据实际需求调整
